🐍 ¿Qué se puede hacer en Python?

Ejemplos comunes de código en Python

1. Variables y tipos de datos:

x = 10 nombre = "Python" pi = 3.1416

2. Condicionales:

edad = 20 if edad >= 18: print("Eres mayor de edad") else: print("Eres menor de edad")

3. Bucles:

for i in range(5): print(i)

4. Funciones:

def saludar(nombre): print(f"Hola {nombre}") saludar("Ana")

5. Listas:

frutas = ["manzana", "banana", "naranja"] print(frutas[0])

6. Diccionarios:

persona = {"nombre": "Luis", "edad": 30} print(persona["nombre"])

7. Módulos:

import math print(math.sqrt(16))

8. Entrada del usuario:

nombre = input("¿Cómo te llamas?") print("Hola", nombre)

9. Manejo de errores:

try: numero = int(input("Ingresa un número: ")) print(100 / numero) except: print("Hubo un error")

Estos son solo algunos ejemplos básicos del poder y simplicidad de Python. Aprenderlos te permitirá crear programas útiles, interactivos y cada vez más complejos.
